| Critère                | Angular                | React                   | Vue.js                  |
| ---------------------- | ---------------------- | ----------------------- | ----------------------- |
| Type                   | Framework complet      | Librairie UI            | Framework progressif    |
| Langage principal      | TypeScript             | JavaScript (JSX)        | JavaScript              |
| Structure              | MVC complet            | Composants indépendants | Composants + directives |
| Outils officiels       | CLI, RxJS, Router      | Create React App        | Vue CLI                 |
| Data Binding           | Bidirectionnel         | Unidirectionnel         | Bidirectionnel          |
| Courbe d’apprentissage | Élevée                 | Moyenne                 | Facile                  |
| Taille du bundle       | Lourde                 | Moyenne                 | Légère                  |
| Idéal pour             | Applications complexes | Interfaces dynamiques   | Petits à moyens projets |

+ Avantages Angular :

Structure complète et solide
Support TypeScript
Idéal pour grandes applications

+ Inconvénients :

Plus complexe
Lourdeur initiale

+ React/Vue :

Plus flexibles et simples
Plus léger pour petits projets
